# 🛡️ InvoVault
### *Secure AI Invoice Manager*

InvoVault is a invoice management platform designed to help micro-SMEs and freelancers track their finances.
 
---
- **Dashboard Preview**

![InvoVault Dashboard](https://i.ibb.co/C31HMbMq/Capture.jpg)

## ✨ Features
InvoVault is built to simplify invoice workflows by providing:
- 📊 Dashboard with key metrics  
- 🧾 Create & manage invoices  
- 👥 Client management  
- 💳 Track payment status  
- 📄 Download invoices as PDF  
- ⚡ Fast and responsive UI  
- 🎨 Clean, modern design system  

---
## 🛠 Tech Stack

- **Core**: Next.js 16 (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS v4, Framer Motion (Animations), Lucide Icons
- **Backend**: Supabase (PostgreSQL, Auth, RLS)
- **State Management**: React Context (Custom Theme System)

## 🛡 Security
InvoVault implements **Row Level Security (RLS)** in Supabase, ensuring that users can only view and manage their own invoices, clients.
